Общие положения

Настоящая Политика двух личностей персональных данных описывает, как 'Наши задачи' собирает, использует и защищает вашу личную информацию при использовании нашего приложения для планирования задач.

Используя наше приложение, вы соглашаетесь с условиями сбора и использования информации в соответствии с этой политикой.

Собираемая информация

При использовании нашего приложения мы можем собирать следующую информацию:

  • Созданные вами задачи и связанные с ними данные
  • Даты выполнения и статус задач
  • Информация об устройстве и браузере
  • Данные об использовании приложения (анонимные)

Хранение и защита данных

Все данные хранятся локально на вашем устройстве с использованием современных технологий шифрования. Мы не передаем ваши персональные данные третьим лицам.

Приложение использует локальное хранилище браузера для сохранения ваших задач, что обеспечивает максимальную конфиденциальность.

Использование файлов cookie

Наше приложение не использует файлы cookie для отслеживания пользователей. Мы можем использовать локальное хранилище браузера исключительно для функциональности приложения.

Ваши права

Вы имеете право:

  • Знать, какие ваши данные хранятся в приложении
  • Удалить все данные через функцию очистки в приложении
  • Экспортировать свои данные в удобном формате
  • Отказаться от хранения данных в любое время

Ссылки на другие сайты

Наше приложение может содержать ссылки на другие сайты. Если вы переходите по ссылке на сторонний сайт, please note that мы не контролируем эти сайты и не несем ответственности за их политику конфиденциальности.

Изменения в политике

Мы можем периодически обновлять нашу Политику двух личностей. Любые изменения будут опубликованы на этой странице с указанием даты последнего обновления.
